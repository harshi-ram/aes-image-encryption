# aes-image-encryption
An image encryption tool using the Advanced Encryption Standard (AES) algorithm to securely encrypt and decrypt image files. This project shows cryptography, file handling, and a key in order to protect image data from unauthorized access.

Features:
- AES based image encryption
- Secure image decryption using a key

Technologies Used:
- React.js
- Crypto.js

How to Run:
1. Go to the project root directory.
2. Run the command:
   npm start
